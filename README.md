# Design Pair Sessions

**Live site → https://amitdialpad.github.io/design-pair-sessions/**

1:1 pairing sessions for Dialpad designers learning to build with AI tools. Driver/navigator format — like pair programming, but for designers. You work together on real tasks using Claude Code.

## What's in here

- **The sessions** — Three structured sessions: watch, pair, then fly solo
- **The process** — The design rhythm: when to be in Figma, when to be in code, how to evaluate what Claude builds
- **Project IRL** — A real project traced from first prototype to PR, showing the wrong turns, pivots, and specific feedback that shaped it
- **The Beacon toolkit** — Every slash command, agent, skill, and hook explained for designers
- **Getting started** — Quick reference cheat sheet

## What this is not

Not a coding course. You're learning to understand the medium well enough to evaluate output, join technical conversations, and make your case for users. Your design training is the hard part.

